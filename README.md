# Graduation_Project_Log
说明：本库用于作者毕业设计小组记录日志所用。

###目录/文件说明：
 - Leader：老师记录日志。
 - xxx(人名)：学生自己记录日志。
 - Resources：存放公共资源文件。
 - Notice.md：大家张贴布告。
 - README.md：库说明文件。

###团队协作说明：
将本库Fork到自己账号，clone到本地进行编写，编写完成提交到自己账户，最后Pull requests到本库，大家在develop分支编写代码。
